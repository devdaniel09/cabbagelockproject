CabbageLockApp - Prototipo inicial
Estructura MVC en Flutter. Incluye:
- Login con roles (admin, employee)
- Gestión de inventario básica (CRUD en memoria)
- Responsive layout global (sidebar / bottom nav) y header con notificaciones

Instrucciones:
1. Copia esta carpeta a tu workspace.
2. Ejecuta `flutter pub get`
3. Ejecuta `flutter run -d chrome` //para correr en el navegador
